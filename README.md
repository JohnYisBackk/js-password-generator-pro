# 🔐 Password Generator Pro

A customizable password generator built with **HTML, CSS, and Vanilla JavaScript**.
Generate secure passwords instantly and check their strength in real time.



## 🚀 Live Demo

👉 https://johnyisbackk.github.io/js-password-generator-pro/

---

## ✨ Features

* Generate **random secure passwords**
* Adjustable **password length**
* Choose character types:

  * Lowercase letters
  * Uppercase letters
  * Numbers
  * Symbols
* Option to **exclude similar characters** (O, 0, l, 1, I)
* **Password strength meter**
* **Copy to clipboard** with feedback
* **Reset settings** to default
* Clean and responsive UI

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **Vanilla JavaScript (ES6)**

No external libraries or frameworks were used.

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/password-generator-pro.git
```

2. Open the project folder

3. Run `index.html` in your browser
